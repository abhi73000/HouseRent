# HouseRent
Introduction
HouseRent is a Spring Boot application that helps users manage house rental operations. It provides functionalities to add, update, delete, and search for rental properties, as well as manage tenants and leases.

Features
Add, update, delete, and view rental properties.
Manage tenant information.
Track lease agreements.
Search for properties based on various criteria.

Installation
Prerequisites
Java (JDK 17)
Maven (v3.6.3 or higher)

Steps
Clone the repository:

git clone 

cd houserent

Build the project:
mvn clean install

Running the Application
To run the application:

mvn spring-boot:run

Navigate to http://localhost:8080/ to access the application.
